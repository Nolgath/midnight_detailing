# Galeria "O nosso trabalho" (página Sobre)

Coloca aqui as fotos dos trabalhos (`.jpg`, `.jpeg`, `.png`, `.webp` ou `.avif`).
Aparecem automaticamente na galeria da página `/sobre/`, por ordem alfabética —
usa nomes como `01-bmw-interior.jpg`, `02-polimento.jpg` para controlar a ordem.

Dicas: fotos em 4:3 ou paisagem ficam melhor; comprime para < 300 KB (ex.: tinypng.com).
